--- title: "Занятия по теории и практике буддизма" publish_date: "2017-08-11" ---

Дорогие друзья! С 1 сентября в Московском буддийском Центре Ламы Цонкапы открывается новый курс занятий по буддийской теории и практике. Занятия будут проходить вплоть до начала ежегодного всероссийского ретрита по тренировке ума, приходящегося на ноябрьские праздники. Занятия бесплатны и предназначены для всех желающих.

четверг 19:00 Шаматха
пятница 19:00 Занятия для начинающих
суббота 14:00 Семинары по философии
воскресенье Ламрим/Ретриты*

*Занятия по Ламриму и ретриты чередуются. Начало занятий по Ламриму - 14:00.

Занятия для начинающих

Занятия предназначены для тех кто хочет познакомиться с буддизмом Махаяны. Слушатели познакомятся с такими базовыми темами как Четыре благородные истины, закон кармы, Прибежище, бодхичитта, пустота и бессамостность.

Ламрим

Занятия по тексту великого Учителя прошлого Дже Цонкапы "Ламрим Ченмо" - последовательно по всем темам Ламрима. Занятия этого цикла посвящены предварительным практикам.

Занятия по практике однонаправленной концентрации (шаматхе). Темы занятий будут публиковаться заранее.

Ретриты

Однодневные ретриты по практикам нендро и Ламриму. В перерыве между первой и второй сессиями для участников будет организовано вегетарианское питание. Темы ретритов будут публиковаться заранее.

Внимание: участвовать в ретрите могут те, кто достаточно продолжительное время слушает учение и получал наставления по практике от Досточтимого Геше Джампа Тинлея.

Семинары по философии

Семинарские занятия посвящены изучению буддийской философии. Формат занятий предполагает активное участие всех присутствующих, у которых должны быть определенные базовые знания по темам занятий, включая отдельную дополнительную подготовку к каждому занятию. Темы заняний будут публиковаться заранее.
